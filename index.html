// 08 · straight answers

Do I need to know how to code? +

No. If you can open a file and copy-paste commands from a guide, you can run this. The build-along tier exists for the moments you get stuck — so you never stay stuck.

Does it cost money to run? +

The system is a one-time purchase, yours forever. Voice is free in-browser, and you can wire it to free local models (Ollama) for zero API cost. If you'd rather use a cloud model, you pay that provider directly — no middle tax from me.

What data leaves my machine? +

It's local-first. Your notes stay on your laptop. It only reaches out to the specific APIs you choose to connect. No accounts, no telemetry.

Will you keep updating it? +

Yes. You get free updates as I keep building on top of the same system I run every day. New modules land in your download.

Refunds? +

If you open it, follow the guide, and it doesn't run for you — email me and I'll make it right or refund you. I want operators, not regrets.
